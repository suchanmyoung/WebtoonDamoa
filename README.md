# 📖DamoaWebtoon Project!

### 🔅재밌는 웹툰을 공유하고 의견을 나누는, 다모아웹툰 프로젝트!
* ***
#### 프로젝트를 시작하기 전 스스로에게 하는 약속
*첫째, 꾸준히 리팩토링 할 것입니다.*

*둘째, 테스트 케이스를 작성할 것입니다.*

*셋째, 예외 처리를 신경쓸 것입니다.* 

*넷째, 이해 없는 코드 복붙을 하지 않을 것입니다.* 

***오래 걸리더라도 품질 좋은 클린 코드를 작성하는 것에 집중할 것입니다.***
